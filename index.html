Learn more about TNR and how to help stop cat overpopulation.

There are many cats in the world. As more cats populate cities and suburbs, the less resources shelters and vet clinics have to help our feline friends. To make sure vets, shelters, and fosters have enough resources, learn more about Trap, Neuter, Return nonprofits and see how you can help solve cat overpopulation!
